# Project Overview
Performed end-to-end exploratory data analysis on an e-commerce sales dataset using Jupyter Notebook and Python.<br>
Analyzed customer purchasing behavior, sales trends, shipping performance, cancellations, and fulfillment operations.<br>
Generated business insights and recommendations to improve profitability and operational efficiency.<br>
# Dataset Information

Dataset Type: E-Commerce Sales Data<br>
Features Included:<br>

Product Categories<br>
Shipping Information<br>
Fulfillment Status<br>
Customer Locations<br>
Revenue Data<br>
# Data Cleaning

Converted date columns into datetime format<br>

# Exploratory Data Analysis
Revenue growth analysis over time<br>

# Key Insights
Revenue showed strong growth during mid-2022<br>
“Set” and “Kurta” were the highest-selling categories<br>
Average Order Value (AOV): ₹692<br>
Bengaluru, Hyderabad, and Mumbai generated maximum revenue<br>
Cancellation Rate: 14.22%<br>
Business is primarily B2C-driven (99.27%)<br>
Expedited shipping was preferred over standard shipping<br>
Amazon fulfillment handled the majority of orders<br>
# Recommendations
Focus marketing on high-performing categories<br>
Reduce cancellations through better order management<br>
Increase AOV using combo offers and cross-selling<br>
Improve logistics and delivery optimization<br>
Strengthen inventory forecasting strategies<br>
Enhance customer retention and loyalty programs<br>
